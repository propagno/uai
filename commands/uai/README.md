# UAI Command Spec

Os arquivos `uai-*.md` neste diretório são a fonte canônica dos comandos do UAI.

- Não edite manualmente os adapters gerados em `.claude/`, `.cursor/`, `.github/` ou `.agents/`.
- Gere ou valide a camada multiagente com `uai-cc sync-commands` ou `npm run sync:commands`.
- Os workflows compostos vivem em `workflows/`.
